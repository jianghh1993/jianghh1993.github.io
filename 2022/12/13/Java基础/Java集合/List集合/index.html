二、List实现类

实现类 特点 初始容量 扩容因子 其他
ArrayList 底层基于数组；线程不安全；查询修改快； 10 0.5，不可设置
LinkedList 底层基于双向链表；线程不安全；新增删除快；实现了Queue接口
Vector 线程安全（synchronized修饰）；扩容成倍增长；其他同ArrayList； 10 1，可设置
